Creating/adding a new resource item

Adding a new folder:

By default, your Resources has one folder in it, named after your course or project site. To add a subfolder, next to the existing folder, from the Add menu, choose Create Folders.

To add another folder, from the Create Folders screen, click Add another folder. For each new folder, enter a title. To enter an optional description and to edit access settings, click Add details for this item. If you would like to remove a folder from the list of folders to add, click the red X.

Adding resource items:

To create a new resource item, in the menubar, click Resources. Then, next to the folder in which you want to create the item, click the Add menu, and follow the appropriate steps below:

File upload: To upload an existing document or other file from your computer, follow these steps:

  1. From the Add menu, select Upload Files.

  2. For each resource, click the Browse button, select a file to upload, and then click Open (or you can type the name of the file in the text box provided). You can upload only 20MB at a time, so you may need to upload large files one at a time.

    Note: Filenames that include certain characters cannot be accepted. For a list of unsupported characters, see Why am I having trouble uploading files?

  3. Type a title for the resource (if you don't add a title, the filename will be used for the title), and add an optional description.

  4. To select additional options, click Add details for this item. Then, select from the following options:

    • Choose your file's copyright status, add copyright information, and display a copyright alert.
    • Under "Availability and Access", control who can access the file and when its available, as follows:
      • To display the file to site members only, select Only members of this site can see this folder and its contents.
      • To share the file with another site to which you belong (e.g., as an attachment to a syllabus item for another course or project you're leading), select This folder and its contents are publicly viewable. (This option is not available if the folder to which you're adding the file is publicly viewable.)
      • To display the file only to selected groups, select Display this folder and its contents to selected groups only., and then select the group(s) that should have access to the file. For more information on groups, see Managing groups.
      • To show or hide your file, or set a beginning and/or ending date for the file's visibility, click the appropriate selections. (This option does not appear when you're adding an item to Resources in My Workspace.) Site administrators will always be able to see hidden items, even when they are hidden from other users.

  5. Next to "Email Notification", specify whether or not you want to have members of the site notified automatically via email when the resource is posted. (This option does not appear when you're adding an item to Resources in My Workspace.

  6. Click Upload Files Now to finish.

URL (link to web site): To specify a URL to a web site that will be the resource, follow these steps:

  1. From the Add menu, select Add Web Links (URLs).

  2. For each resource, type the URL in the box. (You may omit the http:// prefix.)

  3. Add a title for your URL link resource. If you don't add a title, the URL will be used as the title of the resource.

  4. To select additional options, click Add details for this item. Then, select from the following options:

    • Choose your URL's copyright status, add copyright information, and display a copyright alert.
    • Under "Availability and Access", control who can access the URL and when its available, as follows:
      • To display the URL to site members only, select Only members of this site can see this folder and its contents.
      • To share the URL with another site to which you belong (e.g., as an attachment to a syllabus item for another course or project you're leading), select This folder and its contents are publicly viewable. (This option is not available if the folder to which you're adding the URL is publicly viewable.)
      • To display the URL only to selected groups, select Display this folder and its contents to selected groups only., and then select the group(s) that should have access to the URL. For more information on groups, see Managing groups.
      • To show or hide your URL, or set a beginning and/or ending date for the URL's visibility, click the appropriate selections. (This option does not appear when you're adding an item to Resources in My Workspace.) Site administrators will always be able to see hidden items, even when they are hidden from other users.

  5. Next to "Email Notification", specify whether or not you want to have members of the site notified automatically via email when the resource is posted. (This option does not appear when you're adding an item to Resources in My Workspace.)

  6. Click Add Web Links now to finish.

Citation list: To create and display a citation list, follow these steps:

  1. From the Add menu, select Add Citation List.

  2. Choose Create New Citation.

  3. Next to "Select Citation Type", use the drop-down list to choose from the following:

    • Journal Article
    • Book
    • Book Section
    • Report
    • Unknown

  4. Enter all of the appropriate information for the citation, and then click Add another to add another citation, or Save Citation to save the citation.

  5. On the Revise Citation List page, make any needed changes to the list (e.g., revise an existing citation, add citations to the list, or remove citations from the list), and click Finish.

  6. Enter a title and optional description in the appropriate text areas.

  7. Choose your citation list's copyright status, add copyright information, and choose whether or not to display a copyright alert.

  8. Under "Availability and Access", control who can access the text document and when its available, as follows:
    • To display the citation list to site members only, select Only members of this site can see this folder and its contents.
    • To share the citation list with another site to which you belong (e.g., as an attachment to a syllabus item for another course or project you're leading), select This folder and its contents are publicly viewable. (This option is not available if the folder to which you're adding the citation list is publicly viewable.)
    • To display the citation list only to selected groups, select Display this folder and its contents to selected groups only., and then select the group(s) that should have access to the citation list. For more information on groups, see Managing groups.
    • To show or hide your citation list, or set a beginning and/or ending date for its visibility, click the appropriate selections. (This option does not appear when you're adding an item to Resources in My Workspace.) Site administrators will always be able to see hidden items, even when they are hidden from other users.

  9. Next to "Email Notification", specify whether or not you want to have members of the site notified automatically via email when the resource is posted. (This option does not appear when you're adding an item to Resources in My Workspace.)

  10. Click Finish to add the citation list to Resources.

Form items: To create a form item for use in ePortfolio, follow these steps:

  1. From the Add menu, select New Form Item, and under "Select a form", select one of the following:

      2. Add a comment in the text box.
      3. Click Continue.
    • Expectation
      1. Add content to the "Evidence" and "Connect" text boxes.
      2. Click Continue.

      1. Add a comment in the text box.

      1. Add content to the "Intellectual Growth" text box.

Editing a resource item

The edit option allows you to edit a resource from within the Resources tool. If you want to edit an existing resource, you can either revise it or replace it, depending on the type of resource. You can revise plain text and HTML documents; for other document types, the edit option only allows you to revise the information about the resource (e.g., the description), or replace the document's contents (e.g., replace a picture or URL). You can only replace document content with documents you upload from your own computer, one at a time.

To edit a resource:

  1. If it is a document type for which you can revise the content, you will see a drop-down list labeled "Content:" from which you should choose either Revise document contents below or Replace with content from my computer. If it's a document type for which you can only replace content, click the Browse... button beside "Replace with this file:".

  2. For either option, alter any information about the resource you'd like to change as well, including the optional properties. (To view the optional properties, click the arrow beside "Optional properties".)

Deleting a resource item

Use the checkboxes to select the resource items you want to delete, and then click Remove Checked. A confirmation screen will appear asking if you are sure you want to delete the item. Click Remove to confirm, or click Cancel.
